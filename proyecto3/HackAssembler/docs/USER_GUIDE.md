# Guía de uso

## Lo que necesitas
- Python 3

## Cómo correrlo
Desde `proyecto3/HackAssembler/src`:

```bash
python3 HackAssembler.py Prog.asm
```

> Importante: `Prog.asm` debe existir antes de ejecutar el comando.  


## Qué debería pasar
- Si todo sale bien, no imprime nada y se genera el archivo .hack correspondiente.
- Si hay error, te dice la línea y el problema.
